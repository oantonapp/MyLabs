API Reference (MyLabs/backup_mon)

Backup_mon.sh

This is main module

MAIN_Backup_mon.sh

Environment variables

Var : backup_directory

Set backup directory

Environment variables

Var : backup_tag

Set backup tag

Environment variables

Var : log_path

Set log path

Environment variables

Var : conserved_vds_quantity

Set the number of conserved VDS per month Set 0 (zero) if you do not delete anything

Environment variables

Var : used_pool

USE POOL or Standalone

Parameters

bool : true or false

Environment variables

Var : check_day

Checking the day of the week

Parameters

bool : true or false

Environment variables

Var : day_of_week

Set the day of the week when backup is enabled

template: “Sat”

Environment variables

Var : backup_vms

Which VMs to backup. Possible values are:

“all” - Backup all VMs

“running” - Backup all running VMs

“list” - Backup all VMs in the backup list (see below) or existing backup tag

“none” - Don’t backup any VMs, this is the default

“tag” - Backup VMs whith backup tag
